# Quick Start Guide - StaticWebProjDemo

## 🚀 Fastest Way to Run

### Windows (PowerShell)
```powershell
cd c:\Users\Admin\JAVA_VISEM\JFS_VISEM\public
Start-Process test.html
```

### With Local Server (Better)
```powershell
cd c:\Users\Admin\JAVA_VISEM\JFS_VISEM\public
python -m http.server 8000
# Then open: http://localhost:8000
```
